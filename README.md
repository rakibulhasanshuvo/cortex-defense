# Cortex Defense - AI-Powered Landing Page

A high-fidelity, interactive landing page for a decentralized security protocol, built with **Next.js 16+**, **Tailwind CSS**, and **Framer Motion**.

## 🚀 Experience
- **Fluid Interactivity**: Custom magnetic buttons and 3D tilt effects.
- **Atmospheric UI**: Dynamic custom cursor and mouse-following ambient glow.
- **Optimized Architecture**: Next.js App Router for 2026-level performance.
- **Glassmorphism**: Premium deep-blur glass components with gradient borders.

## 🛠️ Tech Stack
- **Framework**: Next.js 16.0.0 (App Router)
- **Styling**: Tailwind CSS v4.0
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 🌐 Deployment
This project is pre-configured for **Vercel**. Simply push to GitHub and import the repository into Vercel. The root `package.json` and `src/` directory are optimized for zero-config deployment.
